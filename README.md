### 对照表：
|  url 传参  |             对应作用             |  默认值  |
| :--------: | :------------------------------: | :------: |
|    name    |             应用名称             |    无    |
|  country   |   国家/地区 (cn,us,jp,kr... )    |    cn    |
|   entity   | 搜索对象 (software/macSoftware ) | software |
|   limit    |      搜索数量限额 (1..200 )      |    18    |
|    cut     |       是否裁切圆角 (0/1 )        |    1     |
| resolution |      分辨率 (256/512/1024 )      |   512    |
|   format   |     图片格式 (jpeg/png/webp )     |   png    |
